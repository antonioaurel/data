Conexões da História

Um mapa interativo das interconexões entre locais históricos, personagens e eventos que moldaram o Recife e Pernambuco. Descubra como lugares e momentos aparentemente desconexos estão profundamente ligados através do tempo.

Com centenas de links informacionais, o Conexões da História é um mapa da fascinante ligação entre locais, eventos e personagens que definiram o nosso presente e influenciam o nosso futuro.

Este projeto utiliza a base interconectada do Recife como laboratório de visualização de dados e qualidade de dados — mapeando lacunas, inconsistências e oportunidades de curadoria no acervo.
Fontes principais: Trilhas do Recife, Ruas do Recife e Pernambucanidade.

Diagrama de nós

Mapa de conexões interativo

Grafo de força com 564 nós e 756 conexões. Cada círculo é uma entrada; as linhas representam relações históricas documentadas.

Como usar
  • ⊕
    Roda do mouse para zoom; arraste o fundo para navegar
  • ○
    Cor indica categoria: azul = Local, laranja = Personagem, verde = Evento
  • ✦
    Clique num nó para ver descrição, imagem e conexões no painel
  • ⊙
    No painel, clique numa conexão para saltar diretamente a ela
  • ⠿
    Arraste nós individualmente para reorganizar o layout
Matrix de adjacência

Mapa completo de todos os pares

Visualização 564×564 de todos os pares de nós. Cada ponto dourado é uma conexão entre dois registros históricos.

Como usar
  • ▦
    Ponto dourado = conexão entre a linha e a coluna
  • ⊕
    Botões + / − ou Ctrl + roda do mouse para zoom
  • ✦
    Clique numa linha ou célula para destacar as conexões daquele nó
  • 🔍
    Barra de busca para encontrar e centralizar um nó
  • ↗
    No painel, clique numa conexão para navegar até ela
Qualidade dos dados

Fill Rate do Schema

Bullet chart mostrando a cobertura de cada campo dos 564 registros: total de preenchidos, total de vazios e percentual. Utilizado para monitorar a qualidade dos dados usados e orientar ações corretivas de curadoria.

Como usar
  • ▌
    Barra âmbar = registros preenchidos; fundo claro = registros vazios
  • │
    Linha vertical cinza = meta de 80% de preenchimento
  • ≡
    Campos core (Nome, Tipo, Local…) e interconexões em seções separadas
